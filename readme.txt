#دستورات نصب 
unzip Dmnnd.zip
cd Dmmnd
chmod +x Dmnd
./Dmnd install
صبر کنید تا نصب شه 

سپس در پوشه config ایدی عددی سودو و ایدی کانال خود را جایگزین کنید 

برا ساخت ربات جدید
./Damnd create
بعدش شماره ی ربات ، api id و api hash را وارد میکنید و در پایان شماره ربات 

برا لانچ ربات از دستور زیر استفاده کنید
cd Dmnd 
tmux
screen ./Dmnd launch شماره ربات
